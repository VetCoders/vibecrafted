Vibecraft

Craft → Converge → Ship

Vibecrafting is what happens when you stop treating AI like a magic wand and start treating it like a stochastic engine that needs iron-clad guardrails. Shape the path, loop until it stops breaking, then force it to ship.

Craft

Stop blind-prompting. Force the agents to read the AST, index the history, and respect the project's actual structure before writing a single line.

Converge

LLMs generate noise. Accept it. Run Marbles loops to systematically crush the entropy until P0=0 and the system actually runs.

Ship

Working code is worthless if nobody can buy it. The final phase forces the system out of the "Always-in-Production" plague and into the market.

The Suite

What's in the Suite

Marbles

Denoise Until the Circle Is Full

AI agents are stochastic parrots. Every time they generate code, they introduce noise. You cannot prompt your way out of it. You have to iterate, measure the remaining entropy, and reduce it.

The Isomorphism

Image diffusion:  noise → denoise × N → image
Code diffusion:   chaos → reduce_entropy × N → product

Step 0: Pure noise (no context, no structure)
Step 1: Init — gross shapes emerge
Step 2: Implement — detail generation
Step N: Converged — DoU < threshold = DoD

There is no fixed loop count. The loop runs until P0=P1=P2=0 — the circle is full. A trivial task may converge in 1 loop. A massive architectural change may take many more. Like learning rate decay, early loops make large corrections (P0 blockers), late loops make small refinements (P2 polish).

The Definition of Undone

AI agents have solved the hardest part of software creation: making the thing work. They have not yet solved the second hardest part: making the thing reachable.

The Always-in-Production Plague — technically excellent software that never ships to users. The cause is not bad code. It's the systematic gap between "it works" and "someone can find, install, and pay for it."

The Plague Diagnostic:

  • + Tests pass
  • + Architecture is sound
  • + README exists
  • – Nobody can find it from Google
  • – Nobody can install it without a toolchain
  • – Nobody can pay for it

Items 4–6 are the Definition of Undone. Phase 1 crafts the system into shape. Phase 2 converges it. Phase 3 ships it complete.

Quick Install

curl -fsSLO https://raw.githubusercontent.com/VetCoders/vc-skills/main/install.sh && bash install.sh

Fast path: download the installer first, then execute it locally. It sets up the central store in ~/.agents/skills and wires up the aliases. Want the Zsh helpers so you can type codex-implement instead of raw shell scripts? Add --with-shell.

On Linux with GNU coreutils, use sha256sum -c install.sh.sha256 instead of shasum -a 256 -c.
