The Insight

Code Follows Diffusion Dynamics

AI agents generate code stochastically. Each generation introduces signal AND noise. The only way to separate them: iterate, measure, reduce.

The Isomorphism

Image diffusion:  noise → denoise × N → image
Code diffusion:   chaos → reduce_entropy × N → product

Step 0: Pure noise (no context, no structure)
Step 1: Init — gross shapes emerge
Step 2: Implement — detail generation
Step N: Converged — DoU < threshold = DoD

There is no fixed loop count. The loop runs until P0=P1=P2=0 — the circle is full. A trivial task may converge in 1 loop. A massive architectural change may take many more. Like learning rate decay, early loops make large corrections (P0 blockers), late loops make small refinements (P2 polish).

The Manifesto

The Definition of Undone

AI agents have solved the hardest part of software creation: making the thing work. They have not yet solved the second hardest part: making the thing reachable.

The Always-in-Production Plague — technically excellent software that never ships to users. The cause is not bad code. It's the systematic gap between "it works" and "someone can find, install, and pay for it."

The Plague Diagnostic:

  • + Tests pass
  • + Architecture is sound
  • + README exists
  • – Nobody can find it from Google
  • – Nobody can install it without a toolchain
  • – Nobody can pay for it

Items 4–6 are the Definition of Undone. Phase 1 builds it right. Phase 2 converges it. Phase 3 ships it complete.

Get Started

Installation

Bootstrap the Suite

# Bootstrap repo + install skills + helper layer
bash install.sh --checkout "$HOME/.local/share/vetcoders-skills" --with-shell

# Or install only the optional zsh helper layer later
bash vetcoders-spawn/scripts/install-shell.sh

Requirements

# Hard foundations
cargo install aicx loctree-mcp

# Optional specialist tool for PR review
cargo install prview

# Then add the agent CLIs you actually use
# codex, claude, gemini

Quick Start

# The one-liner — full pipeline
> "Ship: add auth module with JWT"

# Or run individual phases:

# 1. Initialize context
> "Init session for this repo"

# 2. Run full pipeline
> "ERi pipeline for adding auth module"

# 3. Audit implementation
> "Follow-up check"

# 4. Loop until converged
> "Marbles — loop until clean"

# 5. Check product surface
> "DoU audit — are we shippable?"

# 6. Package for market
> "Hydrate this project"
